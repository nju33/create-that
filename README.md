## Gitpod variables

- `GIT_USER_SIGNINGKEY` is a sign necessary when you do `git commit`
- `NGROK_AUTHTOKEN` is ngrok's authentication
- `GPG_SECRET_KEY_NJU33` is nju33's secret key
- `GH_AUTHTOKEN` is GitHub CLI's authentication
